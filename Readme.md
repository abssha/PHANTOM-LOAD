# ⚡ Phantom Load

A smart home energy audit app that tracks electricity usage and provides AI-powered insights.

## 🚀 Features

* Room-based appliance tracking
* Real-time energy + cost calculation
* Configurable electricity rate
* AI Energy Advisor (Gemini-powered)

## 🛠️ Tech Stack

* Frontend: React + Vite
* Backend: Node.js + Express
* Database: MongoDB
* AI: Google Gemini

## 📁 Project Structure

TeamError/
├── backend/
├── frontend/

## ⚙️ Setup

### Backend

cd backend
npm install
npm run dev

### Frontend

cd frontend
